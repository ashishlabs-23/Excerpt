# Design System: Excerpt
**Project ID:** 13194884443124485003

## 1. Visual Theme & Atmosphere
Exciting, high-energy, and professional. The aesthetic is deep dark mode with vibrant neon accents, emphasizing speed, viral potential, and high-tech AI processing. Generous whitespace and glassmorphism elements create a premium, layered feel.

## 2. Color Palette & Roles
* **Electric Indigo (#6366f1):** Primary Action color used for CTAs, focus states, and progress indicators.
* **Deep Obsidian (#030712):** Main background color for a high-contrast dark mode.
* **Slate Surface (#111827):** Card and container background for subtle depth separation.
* **Vibrant Mint (#10b981):** Success states and positive metrics (e.g., virality score).
* **Muted Slate (#94a3b8):** Secondary text and icon color for balanced hierarchy.
* **Pure White (#ffffff):** Primary content and heading text.

## 3. Typography Rules
* **Font:** Inter (Modern Sans-Serif).
* **Headings:** Bold and expressive, with tight letter-spacing for a modern look.
* **Body:** Clean and readable with slightly increased line-height (1.6) for clarity.

## 4. Component Stylings
* **Buttons:** Generously rounded (12px) with subtle glows for primary actions. Pill-shaped for secondary links.
* **Cards/Containers:** Subtly rounded (16px) with glassmorphism (backdrop-blur) and whisper-soft borders (#1f2937).
* **Inputs/Forms:** Deep slate background with high-contrast active borders (Electric Indigo).

## 5. Layout Principles
* **Structure:** Centered, balanced layouts with a focus on clear hierarchy.
* **Spacing:** Large vertical padding between sections to maintain an "airy" feel despite the dark theme.
* **Grid:** Standard 12-column grid system for desktop; single column stack for mobile.

## 6. Design System Notes for Stitch Generation
When generating screens for Excerpt, ALWAYS use the following tokens:
- **Theme:** Dark, modern, premium.
- **Primary Color:** Electric Indigo (#6366f1).
- **Background:** Deep Obsidian (#030712).
- **Border Radius:** 12px for buttons, 16px for cards.
- **Font:** Inter (Sans-serif).
